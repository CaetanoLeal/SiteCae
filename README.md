## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/CaetanoLeal/SiteCae.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SiteCae
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
